Desvendando o HTML

Um site em HTML funciona como a base de qualquer página da internet. O HTML (linguagem de marcação de hipertexto) serve para organizar o conteúdo — tipo textos, imagens, botões e links — e mostrar isso pro navegador (como o Chrome ou o Firefox). Quando tu acessa um site, o navegador lê o código HTML e transforma ele em algo visual na tela. Geralmente o HTML trabalha junto com o CSS (que cuida das cores e do estilo) e o JavaScript (que faz as partes interativas). Ou seja: HTML = estrutura, CSS = aparência, JavaScript = ações.
